# Credits & Acknowledgments

## Original Project

This project is based on **[pylecular](https://github.com/alvaroinckot/pylecular)** by **Alvaro Inckot**.

The original pylecular project provided the foundation for MoleculerPy, implementing core Moleculer protocol features in Python. We are grateful for this excellent work that made our project possible.

- **Original Author**: Alvaro Inckot
- **Original Repository**: https://github.com/alvaroinckot/pylecular
- **License**: MIT License

## Inspiration

MoleculerPy is inspired by **[Moleculer.js](https://moleculer.services)** - the fast & powerful microservices framework for Node.js.

- **Repository**: https://github.com/moleculerjs/moleculer
- **Website**: https://moleculer.services
- **Author**: Icebob (https://github.com/icebob)

We aim to bring the same powerful microservices capabilities to the Python ecosystem.
